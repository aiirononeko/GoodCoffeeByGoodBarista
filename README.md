# 概要
バリスタとカフェオーナー向けの転職マッチングサイト

# 機能一覧
・ログイン機能
<br>
・バリスタユーザー一覧表示機能
<br>
・バリスタユーザー詳細表示機能
<br>
・バリスタユーザー登録機能（プロフィール画像アップロード機能）
<br>
・オーナーユーザー一覧表示機能
<br>
・オーナーユーザー詳細表示機能
<br>
・オーナーユーザー登録機能（プロフィール画像アップロード機能）

# 技術一覧
言語: Ruby:2.5.3
<br>
フレームワーク: Ruby on Rails:5.2
<br>
データベース: MySQL
<br>
仮想環境: Docker
<br>
インフラ: AWS
